
# Foret APP VER 2.0
온라인 스터디 모임 어플
- spring boot
- mariaDB
- kotlin

## 참조
**[Foret APP](https://github.com/azqazq195/Project/blob/master/Foret)**

## 구현 기능
- 로그인 및 회원가입
- 가입한 그룹 출력, 페이징 처리, 회원 역할에 따라 데이터 노출 및 상태 변경
- 게시판, 댓글, 대댓글, 선택 정렬, 좋아요 기능
- 사진 업로드 및 수정 삭제
- 포레 만들기 및 수정, 가입하기

## 설계 중점
- 기존 1버전의 개선 어플
- MVVM 패턴 제작
- Retrofit 사용
- Spring Boot로 RestFul API 구현

## 차후 업데이트 및 보완점
- 로그인 토큰
- Firebase 활용 푸쉬 알림, 채팅
